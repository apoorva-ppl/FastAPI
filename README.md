# Patient Management System API

A simple and clean REST API built using **FastAPI** to manage patient records.  
This project demonstrates API routing, path parameters, query parameters, data loading from JSON, sorting logic, and proper error handling.

---

## Features

- Home and About routes
- View all patient records
- View a specific patient by patient ID
- Sort patients based on height, weight, or BMI
- Proper HTTP error handling using status codes
- Clean and modular code structure

---

## Tech Stack

- Python 3
- FastAPI
- JSON (for data storage)
- Uvicorn (ASGI server)
